Latest Update - 2016-08-03

v2.1.0

Welcome to the new version of Bracketor. Made using the Ionic Framework. Now supporting Swiss Style tournaments, tournament view history, bookmarked tournaments, and matches with sets.

Please report any bugs you encounter or any feature requests through GitHub or Twitter. Links are available in the about section.

Changelog:

v2.0.9
- Fix major issue where tournaments could not be viewed or created on older devices. - Fix issue where bracket view would not work on older devices.

v2.0.8
- Added a bracket view tab to the tournament view, edit matches by enabling editing and then clicking on the match in the bracket. - Bulk add participants in the participants tab. - Rotation enabled! - Tournaments now sorted by last updated time.
