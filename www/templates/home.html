Latest Update - 2016-02-04

v2.0.3

Welcome to the new version of Challonger. Made using the Ionic Framework. Now supporting Swiss Style tournaments, tournament view history, bookmarked tournaments, and matches with sets.

Please report any bugs you encounter or any feature requests through GitHub or Twitter. Links are available in the about section.
